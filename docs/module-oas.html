(async, static) loadOas(oasFile, options) → {Promise}

Load swagger and/or OpenAPI specification The specification can be a single file, or that can made of several partials.

Parameters:
Name Type Description
oasFile String

The path of the root file of the API specification.
options Object

The options of the loader. See swagger-parser options for details.
Returns:

A Promise, that resolves to an API descriptor object, that provides inner functions to access to the individual endpoints as well as to the whole loaded model.
